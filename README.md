# Screenshot Data Extractor (Free)

This tool extracts structured data from uploaded screenshots using **free, in-browser OCR**.

## Features

- Upload one or many screenshots at once.
- Extract (if available in screenshot):
  - Name
  - Mobile Number
  - Email ID
  - UPI ID
- Choose which fields you want before extraction.
- Copy tabular output with one click (ready to paste into Google Sheets / Excel with separate columns).
- Download as CSV or Excel (`.xlsx`).

## How to run

Open `index.html` in any modern browser.

No backend, no paid API, no server required.
